SVS Architecture — AR Marker

Print this marker and attach it to your poster.
The 3D architecture appears above it when scanned.

Setup instructions:
1. Print this marker at ~10×10cm on your poster
2. Host svs_ar_webxr.html on GitHub Pages
3. Generate QR code → link to hosted page
4. Jury scans QR → opens camera → points at this marker → 3D architecture appears 🎯

Note: Best in good lighting. Keep marker flat and unobstructed.
